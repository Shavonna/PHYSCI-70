Press Fit Constructions

For this week, I learned how to use the laser and vinyl cutter and made several press fit objects to practice!


Object 1: Milk Carton with Slicer for Fusion

  • For this object, I downloaded a 3d obj model of a milk carton, and then translated that into slicer from fusion. From this exercise alone I learned so much about both fusion and slicer, the fit/allowance of cardboard, and the functionality of the laser cutter as well. My first print fit together pretty well, so I didnt need to do any additional cuts. I was able to have help from a classmate to set the offset to 0.2, which turned out to be perfect for the cardboard. From there I followed the assembly instructions and now I have a cardboard milk carton!

    • Object 2: Mini Triangle Shelf

  • For this object, I wanted to make a small shelf that would hold tiny objects. I did this one because I wanted to start from scratch. However, the first object did not translate well and I was not able to press them together. Then, I edited the kerf and the depth of the slits as well. The second version I made turned out perfectly!
